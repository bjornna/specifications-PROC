=== Intellectual Origins

Many of the design ideas presented in this specification came from industry sources, as well as research literature. The openEHR implementers Marand and DIPS provided significant detailed requirements, use cases, and design ideas. 

Additionally the Activity-Based Design (ABD) project at Intermountain Healthcare (2015-), on which the primary author worked part time proved an important source of cross-fertilisation. The latter project, led by David Edwards and Alan James (chief architect), has a sophisticated architecture based on a custom archetype / template framework, adaptive workflow concepts, application generation and real-time speech recognition. It has pioneered a number of concepts used in the current specification, including the separation of Definition, Materialised and Runtime levels of representation.
